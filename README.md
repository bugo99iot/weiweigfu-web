# WeiWeiGFU

## About

Website for the WeiWeiGFU project.

## Setup

- Create a virtutal environment as `weiweigfu-web/venv` with Python 3.8.12.
- Install requirements: `pip install -r requirements.txt`

## Running the app

- Use make target `start-app-local`.
